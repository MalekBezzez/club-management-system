# Club Management System

A web application for managing clubs, members, and activities built with the MERN Stack.

##  Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js

##  Features

- User authentication (JWT)
- Manage club members (add, update, delete)
- Manage clubs and activities
- Role-based access (admin/user)
- Responsive and component-based frontend built with React.js,
